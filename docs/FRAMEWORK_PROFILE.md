## Profile Descriptions

### Solo Profile

Applies when team size is 1. Activates Engineer, System Architect, Review Council, QA, and Innovator by default. All other agents are available via explicit invocation. Council defaults to 4-perspective Micro Council. All quality guarantees (completion gate, security non-negotiable, no false-complete) apply identically.

### Small Team Profile

Applies when team size is 2–10. All 12 agents active by default. Full 7-perspective council. No audit trail overhead. Suitable for startups and product teams.

### Enterprise Profile

Applies when team size is 10+ or the project is in a regulated industry. All 12 agents active. Full 7-perspective council is the mandatory default (no automatic downgrade to Micro Council). Audit trail enabled. Compliance audit prompt required before regulated PRs. CODEOWNERS file recommended (`docs/templates/CODEOWNERS.template`).

**Enterprise teams**: after bootstrapping, also complete these steps:
1. Copy `docs/templates/CODEOWNERS.template` to `.github/CODEOWNERS` and assign your team members.
2. Add the CI audit gate from `docs/templates/ci-quality-gates.template.yaml` to your workflows.
3. Run `.github/prompts/compliance-audit.prompt.md` on each regulated PR and commit the record to `docs/audit/`.
